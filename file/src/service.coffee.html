~ (void) _get_or_fetch(propName, store, path, key, cb) Private

A private helper for a repeated pattern where we only fetch a piece of information if it is not already available in a instance or static cache.

Parameters:

  • propName ( String ) — The name of this property. A promise will be set at this property.
  • store ( Object ) — The cache of things to check if we can use a cache.
  • path ( String ) — The path to GET from if we have to make a request.
  • key ( String ) — The property of the JSON response which has the value that should be yielded to the user.
  • cb ( -> ) — A callback that accepts this kind of thing. (optional)

~ (Promise<BufferedReader<Array<Object>>>) Service(q, args...)

Process the results of a query row by row.

Parameters:

  • q ( Query ) — The query to run.
  • page ( Object ) — The page of results to return.
  • doThis ( -> ) — A callback for each row (optional).
  • onErr ( -> ) — A callback to handle errors (optional).
  • onEnd ( -> ) — A callback to be called when all rows have been received (optional).

Options Hash: (page):

  • start ( Number ) — The index of the first row to return (optional; default = 0)
  • size ( Number ) — The maximum number of results to return (optional; default = null, ie. all)

Returns:

  • ( Promise<BufferedReader<Array<Object>>> ) — a promise to yield an iterator over the rows.

Static method to flush the cached models, versions and summary-field informations.
